# CodeAlpha Data Analytics Internship

This repository contains the data analytics projects completed as part of my **Data Analytics Internship at CodeAlpha**.

## 📊 Projects

### 1. Exploratory Data Analysis (EDA) — IPL Match Dataset
Performed in-depth exploratory data analysis on IPL match data to uncover trends, patterns, and key insights using Python.

**Tools Used:** Python, Pandas, Matplotlib, Seaborn

---

### 2. Data Visualization — Netflix Content Analysis
Analyzed Netflix's content library and created visualizations to understand trends in genres, release years, and content types.

**Tools Used:** Python, Pandas, Matplotlib, Seaborn

---

### 3. Sentiment Analysis — Amazon Product Reviews
Built a sentiment analysis model to classify Amazon product reviews as Positive, Negative, or Neutral using NLP techniques. Also generated a word cloud to visualize the most frequently used words in customer reviews.

**Key Insight:** 98.2% of reviews were Positive, indicating high customer satisfaction.

**Tools Used:** Python, Pandas, TextBlob, WordCloud, Matplotlib

---

- Natural Language Processing (NLP)
